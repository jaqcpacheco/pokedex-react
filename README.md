# 🎮 Pokédex React

Uma aplicação web interativa desenvolvida com React que consome dados da PokeAPI e exibe Pokémons de forma dinâmica, responsiva e moderna.

---

## 🚀 Tecnologias Utilizadas

- React
- Vite
- Tailwind CSS
- Axios
- JavaScript
- PokeAPI

---

## 📡 API Utilizada

### PokeAPI

API pública utilizada para obter os dados dos Pokémons.

🔗 https://pokeapi.co/

### Endpoint utilizado

```bash
https://pokeapi.co/api/v2/pokemon
```

---

## ✨ Funcionalidades

✅ Listagem dinâmica de Pokémons  
✅ Sistema de busca em tempo real  
✅ Componentização com React  
✅ Consumo de API REST  
✅ Gerenciamento de estado com useState  
✅ Requisições com useEffect  
✅ Custom Hook reutilizável  
✅ Formulário controlado  
✅ Interface responsiva com Tailwind CSS  
✅ Cards interativos com animações

## ⚙️ Como Executar o Projeto

### Clone o repositório

```bash
git clone https://github.com/jaqcpacheco/pokedex-react.git
```

---

### Acesse a pasta do projeto

```bash
cd pokedex-react
```

---

### Instale as dependências

```bash
npm install
```

---

### Execute o projeto

```bash
npm run dev
```
